# 用户信息

**/etc/passwd**

- 存储条目格式

    ```
    name:coded-passwd:UID:GID:userinfo:homedirectory:shell

    例子
    root:x:0:0:root:/root:/usr/bin/bash
    ```

    - 若用户ID=0，则表示该用户拥有超级用户的权限
    - 检查是否有多个ID=0
    - 禁用或删除多余的账号

**/etc/shadow**

- 存储条目格式

    ```
    name:passwd:lastchg:min:max:warn:inactive:expire:flag

    例子
    daemon:*:17834:0:99999:7:::
    ```
